# Quadratic Formula

**Source:** College Algebra 2e, Chapter 2

## The Problem

Solve any quadratic equation of the form ax² + bx + c = 0 where a ≠ 0.

## The Formula

```text
x = (-b ± √(b² - 4ac)) / 2a

Discriminant: Δ = b² - 4ac
  Δ > 0 → two real solutions
  Δ = 0 → one real solution (repeated root)
  Δ < 0 → two complex solutions
```

## Derivation (Completing the Square)

```text
ax² + bx + c = 0
x² + (b/a)x = -c/a
x² + (b/a)x + (b/2a)² = -c/a + (b/2a)²
(x + b/2a)² = (b² - 4ac) / 4a²
x + b/2a = ±√(b² - 4ac) / 2a
x = (-b ± √(b² - 4ac)) / 2a
```

## Example

For 2x² + 5x - 3 = 0:
- a = 2, b = 5, c = -3
- Δ = 25 - 4(2)(-3) = 25 + 24 = 49
- x = (-5 ± 7) / 4 = 1/2 or -3
