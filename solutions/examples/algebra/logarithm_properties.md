# Logarithm Properties

**Source:** College Algebra 2e, Chapter 6

## Definition

The logarithm is the inverse of exponentiation:

```text
log_b(x) = y  ⟺  b^y = x

where b > 0, b ≠ 1, and x > 0
```

## Core Properties

```text
Product Rule:   log_b(MN) = log_b(M) + log_b(N)
Quotient Rule:  log_b(M/N) = log_b(M) - log_b(N)
Power Rule:     log_b(M^p) = p · log_b(M)
```

## Special Values

```text
log_b(1) = 0       (since b^0 = 1)
log_b(b) = 1       (since b^1 = b)
log_b(b^x) = x     (inverse property)
b^(log_b(x)) = x   (inverse property)
```

## Change of Base

```text
log_b(x) = log_a(x) / log_a(b)

Common: log_b(x) = ln(x) / ln(b)
                 = log(x) / log(b)
```

## Common Logarithms

- log(x) = log₁₀(x) — common log
- ln(x) = logₑ(x) — natural log, where e ≈ 2.718
